# 3.4 Outros Documentos de Instrução do Despacho

> Parte: **PARTE III — DOCUMENTOS DE TRANSPORTE INTERNACIONAL** · Manual: **Manual de Importação** · Snapshot: 2026-04-27


| Documento | Finalidade |
| --- | --- |
| Invoice (Fatura Comercial) | Comprovante da transação comercial; base para valoração aduaneira. Deve conter: descrição, quantidade, preço unitário, total, Incoterm, dados do comprador e vendedor. |
| Packing List | Lista de volumes; descreve o conteúdo de cada embalagem (peso bruto, líquido, dimensões). Essencial para a conferência física. |
| Certificado de Origem | Comprova o país de origem da mercadoria para fins de aplicação de preferências tarifárias (Mercosul, ALADI) ou controle de origem. |
| Apólice/Certificado de Seguro | Comprova a contratação do seguro internacional (exigido no Incoterm CIF/CIP). Integra o valor aduaneiro. |
| Romaneio de Carga | Documento de controle emitido pelo exportador ou transportador, complementar ao Packing List. |
| Certificado de Análise/Laudo Técnico | Exigido por órgãos anuentes (ANVISA, MAPA, INMETRO) para determinados produtos. |
| DU-E / Documento de Exportação Estrangeiro | Declaração de exportação do país de origem; útil para comprovação de origem e valor nas fiscalizações. |
